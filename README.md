# AI Blog Generator

A multi-agent AI system that generates blog posts automatically using LangGraph.

## Features

- AI Planner Agent
- Research Agent using Tavily Search
- AI Writer Agent
- Blog Reducer Agent
- Streamlit UI

## Tech Stack

- Python
- LangChain
- LangGraph
- Gemini API
- Tavily Search API
- Streamlit
